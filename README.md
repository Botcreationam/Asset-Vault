📚 Acadvault 📚 

Acadvault is an online platform that allows students to access and read academic books and slides directly on the website. The goal is to make learning resources easily accessible while maintaining a controlled system for downloads.

🚀 ** Features**
📖 Read books and slides online
🔐 Secure access to content
💰 Download system using units
🎁 New users get 50 free units
👤 User accounts with profile updates (username, etc.)
💬 Chat system with messaging and replies
📰 Open news feed for posts and interactions

⚙️ How It Works
Users can view/read content for free on the platform
To download files, users must use units
Units are given on signup and can be extended later
Content is protected to prevent unauthorized downloads

🧠 Tech Stack
Frontend: HTML, CSS, JavaScript (or React)
Backend: Node.js / Express (or similar)
Hosting: Replit
Version Control: GitHub

🔒 Security
Restricted file access
Controlled downloads
User authentication system

📌 Status
🚧 This project is currently under development and continuously improving.
